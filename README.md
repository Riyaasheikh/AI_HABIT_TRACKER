# 🤖 AI Habit Tracker

An advanced full-stack habit tracking application powered by artificial intelligence. The app monitors user consistency and leverages the **Gemini AI Engine** to provide contextual performance analysis, weekly reports, dynamic habit suggestions, and adaptive streak recovery roadmaps.

---

## 🚀 Key Features

* **AI Weekly Reports:** Analyzes execution data over the last 7 days to generate a deep-dive performance summary.
* **Smart Habit Suggestions:** Generates 3 highly personalized habit recommendations based on unique user goals, schedules, and past struggles.
* **Streak Recovery Engine:** Detects broken consistency streaks and automatically maps out actionable 3-day recovery roadmaps.
* **Interactive AI Chat:** A context-aware chatbot capable of answering questions about your personal performance metrics over the last 30 days.
* **Morning Motivation:** Delivers personalized motivational text metrics every single morning using actual active streak counts.
* **Interactive UI:** Smooth Drag-and-Drop capability utilizing `@dnd-kit` alongside real-time data visualization charts rendered via `Recharts`.

---

## 🛠️ Tech Stack

### Backend
* **Runtime:** Node.js (v22+)
* **Framework:** Express.js (ES Modules)
* **Database:** MongoDB Atlas + Mongoose ODM
* **AI Engine:** `@google/genai` (Gemini-2.5-Flash)
* **Process Manager:** Nodemon

### Frontend
* **Core Framework:** React 19 + TypeScript
* **Build Tool:** Vite
* **Routing:** React Router DOM v6
* **Styling:** Tailwind CSS v4 + PostCSS
* **Analytics/Charts:** Recharts
* **Date Manipulation:** Date-fns
